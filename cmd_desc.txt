# В этом файле содержатся описания всех команд, используемых в интерфейсу тестирующей системы.
# Эти данные выводятся программой после ввода команды -h.

c login
a имя$пользователя(строка$без$пробелов)
d Авторизует$пользователя.$Пользователь$с$указанным$именем$должен$существовать.

c logout
d Выход$из$системы.

c exit
d Завершает$выполнение$тестирующей$системы.

c create
a имя$пользователя(строка$без$пробелов)
d Создаёт$пользователя$с$указанным$именем.

c start_test
a Опционально:$строка$формата${тип$задания}{количество$заданий$типа}.
a Например,$при$вводе$строки$B5$D4$E8$будет$сгенерированно$5$заданий$типа$B,$4$задания$типа$D$и$8$заданий$типа$E
a Типу$B$соответствуют$задания$на$определение$адреса$подсети$по$IP-адресу$и$маске.
a Типу$D$соответствуют$задания$на$определение$маски$по$IP-адресам$компьютера$и$подсети.
a Типу$E$соответствуют$задания$на$определение$количества$IP-адресов,$соответствующих$условию.
d Генерирует$задания$по$указанной$строке,$или$генерирует$15$заданий,
d используя$статистику$пользователя,$если$строка$не$указана.
d Если$строка$для$генерации$указана,$статистика$по$решённым$заданиям$не$будет$сохранена.

c answer
a Номер$задания$из$подборки$и$ответ$на$задание,$записанные$через$пробел.
d Сохраняет$ответ$на$указанное$задание.

c submit
d Отправлет$ответы,$сохранённые$при$помощи$команды$answer,$на$проверку$и$выводит$отчёт$о$проверке.

c stats(inop)
d Выводит$статистику$пользователя$по$решённым$заданиям.
